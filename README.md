### Vmusic播放器

- 首先VUE全家桶，因为jsonp的关系，还是使用的vue-resource。
- 基于material design, 前端使用部分muse ui的组件，当然其中有一些问题需要解决，因为使用rem的关系，重写了部分样式。
- 为了实现进度滑动效果，修改了部分slider组件的js代码，日后会贴出。
